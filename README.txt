Introduction:
	This application provides aa effective solution to the challenge stated in Programming Challenge.pdf. Please refer the file named - "Programming Challenge.pdf" for more details about the challenge.

Prerequisites:
	.Net Framework 4.7.2 in Visual Studio 2019 - To review the code and execute the application.

Build and run the application:
	1. Download the code from the repository - "ConsumeMagazineStoreAPI" that has source code, README.txt and Programming Challenge.pdf
	2. Open the solution file (ConsumeMagazineStoreAPI.sln) in Visual Studio to open and review the code.
	3. Open NuGet Package Manager console from TOOLS -> NuGet Package Manager -> Package Manager Console and execute following command.
		Install-Package Microsoft.AspNet.WebApi.Client (Latest stable 5.2.7, this also installs Newtonsoft.Json.6.0.4)  
	4. Press F5 to build and run the sample.
